### 1.1 참여과제
주제 : 자유 주제([동국대학교 - CSID DGU](https://github.com/CSID-DGU)에 존재하는 프로젝트/타 오픈소스 프로젝트 의 파생 프로젝트 또는 신규 프로젝트)
